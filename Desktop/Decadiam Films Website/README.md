# Decadiam Films Website

A modern, Apple-inspired website for Decadiam Films - an award-winning film production company based in Sydney, Australia.

## Features

- **Apple-style scrollytelling** with smooth animations
- **Dark theme** with blue accent colors (#007AFF)
- **Responsive design** that works on all devices
- **Video hero section** featuring company intro
- **Professional contact form**
- **SEO optimized** with meta tags and social sharing

## Quick Start

1. **Open the website**: Simply open `index.html` in your web browser
2. **No server required**: The website works as static files

## File Structure

```
Decadiam Films Website/
├── index.html              # Main website file
├── styles.css              # All styling and animations
├── script.js               # JavaScript functionality
├── logo-horizontal.png     # Horizontal logo for navigation/footer
├── logo-vertical.png       # Vertical logo for hero section
├── Decadiam Films intro 2024.mp4  # Hero background video
└── README.md               # This file
```

## Customization

### Adding Your Content

1. **Projects**: Update the project cards in the HTML with your actual film projects
2. **Awards**: Modify the awards section with your real achievements
3. **Contact Info**: Update email and contact details in the contact section
4. **About Text**: Customize the company description

### Adding Project Images

Replace the placeholder project images:
1. Add your film stills/posters to the project folder
2. Update the `src` attributes in the project cards
3. Recommended image size: 500x300px for optimal display

### Contact Form

The contact form currently shows a demo submission. To make it functional:
1. Add a backend service (e.g., Netlify Forms, Formspree)
2. Update the form action in `script.js`
3. Or integrate with your preferred form handling service

## Deployment

### GitHub Pages
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Your site will be live at `username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Instant deployment with custom domain support

### Traditional Web Hosting
1. Upload all files to your web server
2. Ensure `index.html` is in the root directory
